  • تغيير النصوص في الحقول لا يظهر في الـ preview
  • تحديث الـ items لا يظهر في الـ preview
  • الـ live preview لا يتحدث مع تغيير البيانات

🐛 Troubleshooting

If live preview is not updating:

  1. Check browser console for error messages
  2. Verify that pageData is being updated
  3. Check if LivePreview is receiving updated components
  4. Ensure ComponentPreview is transforming data correctly
  5. Verify that AboutJourney is receiving correct props
  6. Check if refreshKey is being updated
  7. Ensure no caching issues
