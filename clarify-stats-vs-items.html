🔍 توضيح الفرق بين Stats و Items

❓ المشكلة المُكتشفة

الخلط بين Stats و Items:

الصورة تُظهر حقل "Statistics" (الإحصائيات) وليس حقل "items". في AboutHeroSection، الحقل الصحيح هو "stats" وليس "items".

📊 بنية AboutHeroSection Schema

AboutHeroSection: { schema: { properties: { title: { type: "string", label: "Hero Title" }, subtitle: { type: "string", label: "Hero Subtitle" }, description: { type: "string", label: "Hero Description" }, backgroundVideo: { type: "string", label: "Background Video URL" }, stats: { // ← هذا هو الحقل الصحيح type: "array", label: "Statistics", items: { type: "object", properties: { value: { type: "string", label: "Statistic Value" }, label: { type: "string", label: "Statistic Label" } } } } } } }

🔍 مقارنة بين Stats و Items

الحقل الوصف الموجود في AboutHeroSection المثال
stats مصفوفة الإحصائيات ✅ موجود
stats: [ { value: "500+", label: "Projects Completed" }, { value: "15+", label: "Years Experience" }, { value: "98%", label: "Client Satisfaction" } ]
items مصفوفة العناصر العامة ❌ غير موجود
items: [ { title: "Item 1", description: "Description 1" }, { title: "Item 2", description: "Description 2" } ]

🎯 ما الذي تُظهره الصورة؟

الصورة تُظهر:
  • ✅ حقل "Statistics" (الإحصائيات)
  • ✅ عناصر "Item 1", "Item 2", "Item 3"
  • ✅ حقول "Statistic Value" و "Statistic Label"
  • ✅ قيم مثل "500+", "15+", "98%"
هذا هو السلوك الصحيح:

AboutHeroSection يجب أن يحتوي على حقل stats لعرض الإحصائيات. الصورة تُظهر هذا الحقل بشكل صحيح.

❓ السؤال المهم

هل تريد:
  1. إزالة حقل stats بالكامل من AboutHeroSection؟
  2. تغيير طريقة عرض حقل stats؟
  3. إزالة حقل items (الذي لا يوجد أصلاً)؟

🔧 الحلول المُقترحة

إذا كنت تريد إزالة حقل stats:
// تعديل schema لـ AboutHeroSection AboutHeroSection: { schema: { properties: { title: { type: "string", label: "Hero Title" }, subtitle: { type: "string", label: "Hero Subtitle" }, description: { type: "string", label: "Hero Description" }, backgroundVideo: { type: "string", label: "Background Video URL" } // إزالة حقل stats } } }
إذا كنت تريد تغيير طريقة العرض:
// تغيير label أو طريقة العرض stats: { type: "array", label: "Key Metrics", // تغيير الاسم // أو إخفاء الحقل تماماً hidden: true }
